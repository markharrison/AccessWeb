Accessibility Statement

Our Commitment to Accessibility

AccessAssist is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Standards

This website aims to conform to the following accessibility standards:

  • WCAG 2.1 Level AA: Web Content Accessibility Guidelines 2.1 at Level AA
  • Section 508: US Federal accessibility requirements
  • EN 301 549: European accessibility standard
  • WAI-ARIA: Web Accessibility Initiative - Accessible Rich Internet Applications

Accessibility Features

For Users with Visual Impairments

  • Voice input and output support using Web Speech API
  • Screen reader compatibility with semantic HTML and ARIA attributes
  • Keyboard navigation for all interactive elements
  • High contrast themes and scalable text up to 250% (XX-Large option)
  • Skip navigation links
  • Alternative text for all meaningful images

For Users with Hearing Impairments

  • Visual alerts for all notifications and status changes
  • Text alternatives for audio content
  • No reliance on sound for critical information

For Users with Cognitive Disabilities

  • Plain language throughout (B1/B2 CEFR level)
  • Clear progress indicators for multi-step processes
  • Consistent navigation and layout
  • Reduced motion options
  • Auto-save functionality to prevent data loss

For Users with Motor Disabilities

  • Large touch targets (minimum 44x44 pixels)
  • Voice input alternatives for text entry
  • No time-limited interactions
  • Keyboard-only navigation support

Testing and Validation

This website has been tested using:

  • Automated testing: axe-core accessibility engine via Playwright
  • Manual testing: Keyboard navigation, screen readers (NVDA, JAWS, VoiceOver)
  • User testing: Testing with users who have various disabilities
  • Color contrast: All text meets WCAG 2.1 AA contrast ratios (4.5:1)
