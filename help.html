üí° Tips for Better Voice Recognition

  • Speak clearly and at normal pace
  • Use natural language - you don't need to be robotic
  • If a command isn't recognized, try a variation from the list above
  • Voice navigation persists across page navigation
  • You can use voice commands while forms and settings are open

üîß Troubleshooting

  1. Check that your browser supports voice recognition (Chrome, Edge, Safari)
  2. Allow microphone access when prompted
  3. Ensure your microphone is not muted
  4. Try refreshing the page and clicking Voice Navigation again

  • Speak more slowly and clearly
  • Try different variations of the command
  • Check if background noise is interfering
  • Ensure your microphone is close enough
  • The system supports both English and Welsh commands
